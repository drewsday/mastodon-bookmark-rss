Mastodon Bookmark RSS

Subscribe to your Mastodon bookmarks as RSS feed

  1. Go to yourinstance.example/settings/applications
  2. Create a new application with permission to read bookmarks
  3. Paste the Access Token back into this form:

About this service

  • This service does not store any data at all. The access token you provide is just encoded in the RSS feed's URL and temporarily stored in server memory every time the RSS feed is requested by your feed reader.
  • This service acts on a user's behalf. It requires explicit setup by a user, like for example a mobile client. Therefore, like a mobile client, this service does not respect robots.txt.
  • This service is open-source, and the code can be found on GitHub. Anybody can run it. Therefore, anybody can modify their installation of it to do things that are not documented here.
  • The corresponding user-agent for this service is of the form mastodon-bookmark-rss/<version> (+<installation_url>). installation_url will point to the public URL of wherever this software is hosted, on a best-effort basis.
