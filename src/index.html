Mastodon Bookmarks RSS

An app to subscribe to your bookmarks as RSS feed.

  1. Go to yourinstance.social/settings/applications
  2. Create a new application with permission to read bookmarks.
  3. Paste the Access Token back into this form:

Privacy

This service does not store any credentials, not even in access logs. The access token you provide is just encoded in the RSS feed's URL and temporarily stored in memory every time the RSS feed is created.
